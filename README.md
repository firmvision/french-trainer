# French Trainer — Road to CLB 7

A single-file, offline French learning web app to take a learner from A2 toward **CLB 7 / B2** for Canadian immigration (TEF/TCF). No install, no server — open `FrenchTrainer.html` in a browser (Chrome or Edge recommended for speech).

## Features
- **Learn / Study** — grammar lessons with examples (présent, passé composé, imparfait, conditionnel, connectors, TEF tips)
- **Verb Lab** — typed conjugation drills focused on past tenses
- **Translation** — EN→FR sentence building with accent-tolerant checking
- **Listening** — French read aloud via browser speech synthesis
- **Dictée** — type what you hear
- **Vocabulary** — themed flashcard decks (incl. immigration & work)
- **Reading** — short passages with comprehension questions
- **TEF-style Mock Test** — timed, mixed exam with a readiness score
- **Placement Test** — estimates your CLB level
- **Profiles** — multiple users, each with their own streak, XP and progress (browser localStorage)
- **Daily reminder** — set a practice time with browser notifications
